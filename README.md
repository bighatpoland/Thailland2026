# Thailand 2026 Travel Companion

Mobilna aplikacja React/Vite z planem podrozy do Tajlandii.

## Deploy na Vercel

Repo jest gotowe do importu w Vercel:

1. Importuj repo: `https://github.com/bighatpoland/Thailland2026`.
2. Vercel uzyje konfiguracji z `vercel.json`.
3. Nie sa wymagane zmienne srodowiskowe.

Konfiguracja deploymentu:

- install command: `cd travel-ui && npm ci`
- build command: `cd travel-ui && npm run build`
- output directory: `travel-ui/dist`

## Lokalnie

```bash
npm run dev
```

Build produkcyjny:

```bash
npm run build
```

Kod aplikacji znajduje sie w `travel-ui/`.
